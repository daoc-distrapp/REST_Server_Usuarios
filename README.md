# REST_Server_Usuarios
Servidor REST

## Notas

Este es un proyecto Spring Boot

## Ejecución

- Alternativa 1
  - En la línea de comando vaya al directorio raíz (aquel con el pom.xml) y ejecute:
    - `mvnw spring-boot:run`
- Alternativa 2
  - En la línea de comando vaya al directorio raíz (aquel con el pom.xml) y ejecute:
    - `mvn clean package`
    - `java -jar target\usrs_rest_srv-0.0.1-SNAPSHOT.jar`
